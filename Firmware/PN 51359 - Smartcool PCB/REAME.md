Unprogrammed Board PN: 51140
Programmed Board PN: 51359

Tool Chain

    Microchip's MPLabX
        MPLabIPE: device programming

    Microchip PicKit3 programmer

Board Programming and Configuration

    Board firmware programming
    1. Power the board with a 12V power supply
	2. Plug the PicKit3 into the 6-pin programming connector
	3. In MPLabIPE, load the firmware image: Schnozzle.X.production.hex
	4. Use MPLabIPE to burn the firmware image
